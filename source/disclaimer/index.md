## 隐私政策

本站唯一涉及隐私的部分为评论系统。

评论系统为自建，需要通过 **GitHub** 或 **Google** 账号登录后才能发表评论。登录时，你需要遵守对应平台的隐私政策：

- [GitHub 隐私声明](https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement)
- [Google 隐私权政策](https://policies.google.com/privacy)

### 我们收集哪些数据（Data We Collect）

当你使用 GitHub 或 Google 登录发表评论时，我们可能会获取并存储以下信息：

- 第三方平台提供的唯一用户 ID
- 显示名称（Name）
- 头像 URL（Profile Picture）
- 邮箱地址（Email）
- 登录时间与评论时间等相关元数据

本站**不收集**你的密码、令牌内容或任何不必要的额外数据。

### 数据如何使用（How We Use Your Data）

这些信息仅用于：

1. 验证登录状态（身份认证）
2. 将评论与你的账号关联
3. 防止垃圾评论和滥用
4. 在评论区显示你的昵称 / 头像

我们**不会**将你的数据用于：

- 广告
- 用户画像（Profiling）
- 向第三方出售或出租数据

### 数据如何存储（Data Storage）

你的数据将存储在位于 **欧盟地区的服务器** 上。

存储内容包括：

- Google/GitHub 提供的账号标识
- 你的评论内容
- 评论的时间戳

不会存储任何第三方平台的访问令牌或敏感授权凭据。

### 数据保留时长（Data Retention）

我们会在以下情况之一时保留你的数据：

- 你的评论仍存在于本站
- 你的账号身份仍用于评论记录
- 法律法规要求保留（若适用）

你可以随时请求删除自己的所有评论记录及关联数据。

### 数据共享（Data Sharing）

我们不会将你的个人数据出售、出租或提供给任何第三方。

仅在以下情况例外：

1. 为提供你请求的服务所必需
2. 应对法律程序、司法要求或政府机关的合法请求
3. 你违反本站条款并需要采取相应措施

本站不参与任何广告网络或跟踪生态。
